# Klerk

Klerk is a 6-day MVP for an AI document processing assistant that extracts structured information from invoices, receipts, and quotes.

## Day 1 goals
- Initialize a TypeScript project with strict configuration
- Add environment validation and structured logging
- Define the core domain model and error handling foundation
- Create the initial Supabase schema for documents

## Getting started
1. Copy `.env.example` to `.env` and fill in the required values.
2. Install dependencies with `npm install`.
3. Build the project with `npm run build`.
4. Start the app with `npm run dev`.
